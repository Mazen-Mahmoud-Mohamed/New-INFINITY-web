# Infinity Web Project

A modern web application with secure user authentication and a beautiful UI.

## Features

- Secure user authentication system
- Modern and responsive design
- Product showcase
- Team information
- Payment integration
- Client logos display

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- bcrypt for password hashing
- Express Session for authentication

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/infinity-web.git
cd infinity-web
```

2. Install dependencies:
```bash
npm install
```

3. Start the server:
```bash
npm start
```

4. Visit `http://localhost:3000` in your browser
